Karel Upper Board
=======
Robotic platform Karel consists of two boards, upper and lower. Upper board contains:
-	Power switch
-	Motor voltage regulator
-	Motor drivers
-	+5V regulator
-	+3.3V regulator
-	Microcontroller
-	User push-buttons (4)
-	User LEDs (4)
-	User potentiometer
-	Bluetooth & Level Shifter
-	ISP, USB, USART interfaces
